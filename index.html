This is the first project. On this page there is a usage of the four requirements for this project.

  • Media Queries
  • Resizing Images with Javascript
  • CSS Sprites
  • Transforming menus from horizontal list items to a vertical drop down menu, or drop down select form control.
The nav at the top when a tablet or mobile size will become into a vertical dropdown list. Media Queries are implemented in the code. All images on the page are being resized accordingly through javascript, other than the background images. Css sprites was used. Via following the demo. and then slightly altered when I was working on my css changes.
When in the mobile view of site, you will notice the images are sized to their actual size, however as the screen becomes smaller, the images resize accordingly via Javascript.
